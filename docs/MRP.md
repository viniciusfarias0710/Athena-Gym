# MRP – Material / Resource Requirements Planning

## Projeto: Site Institucional e Funcional para Academia

---

## 1. Visão Geral do Projeto

**Objetivo do site:**
Criar um site profissional para uma academia, com foco em:

* Divulgação de planos e serviços
* Captação de leads (novos alunos)
* Comunicação com alunos
* Credibilidade da marca

**Escopo considerado no MRP:**
Site institucional com funcionalidades básicas (MVP + evolução).

---

## 2. Estrutura do MRP

O MRP está organizado em **4 categorias principais de recursos**:

1. Recursos Humanos
2. Recursos Tecnológicos
3. Recursos de Informação (Dados)
4. Recursos Financeiros

Cada recurso está associado a:

* Finalidade
* Momento de uso
* Dependências

---

## 7. Planejamento Temporal (MRP x Fases)

### Fase 1 – Descoberta (Semana 1)

* Levantamento de requisitos
* Definição de público
* Arquitetura do site

### Fase 2 – MVP (Semanas 2–4)

* Página inicial
* Sobre a academia
* Planos
* Contato

### Fase 3 – Validação (Semana 5)

* Testes
* Ajustes
* Feedback do cliente

### Fase 4 – Evolução

* Área do aluno
* Agendamento de aulas
* Integração com WhatsApp

---

## 8. Dependências Críticas

* Conteúdo fornecido pela academia
* Aprovação do layout
* Disponibilidade do domínio

---

## 9. Indicadores de Controle (KPIs)

| Indicador             | Meta  |
| --------------------- | ----- |
| Tempo de carregamento | < 2s  |
| Conversão de leads    | > 5%  |
| Disponibilidade       | 99,5% |
| Taxa de rejeição      | < 40% |

---

## 10. Conclusão

Este MRP garante:

* Previsibilidade de recursos
* Controle de custos
* Alinhamento técnico e de negócio
* Base sólida para escalar o site futuramente
